# Eventable

Per-object publish/subscribe events. Mix it into any class and its instances
gain their own event channels — `on`/`emit`/`off`/`count`/`clear` — with LIFO
dispatch, automatic cleanup on destroy, and arbitrary event names. Built on the
core `_Stack` primitive.
